# BLR

## C'est quoi
Régression linéaire bayésienne (Bayesian Linear Regression) : méthode de régression intégrant l'inférence bayésienne pour estimer les paramètres du modèle et leur incertitude. Produit une distribution postérieure sur les coefficients, permettant une quantification explicite de l'incertitude prédictive.

## Papiers
- [[Fraza2021]] — Régression linéaire bayésienne de base (et B-spline BLR) servant de socle et de comparaison.
- [[Ge2024]] — Régression linéaire bayésienne incluse dans le benchmark ; moins précise que les algorithmes optimaux.
- [[Rutherford2022b]] — Régression linéaire bayésienne, algorithme de base recommandé dans le protocole.
- [[Shao2024]] — Régression linéaire bayésienne normative estimée par région.

# NUSDAST

## C'est quoi
NUSDAST (Northwestern University Schizophrenia Data and Software Tool) : jeu de données IRM structurelle en accès ouvert sur la schizophrénie, incluant patients et contrôles sains.

## Papiers
- [[Pinaya2019]] — Échantillon clinique schizophrénie (patients + contrôles) pour le test.

# EVS

## C'est quoi
Théorie des valeurs extrêmes (Extreme Value Statistics) : branche de la statistique qui modélise le comportement asymptotique des maxima ou minima d'un échantillon. Dans la modélisation normative, utilisée pour agréger les déviations régionales en un indice d'anomalie global par sujet (approche block maxima).

# Autisme

## C'est quoi
Trouble du neurodéveloppement (ASD, Autism Spectrum Disorder) caractérisé par des difficultés de communication sociale et des comportements répétitifs ou restreints. Dans le contexte de l'imagerie cérébrale, l'autisme est étudié comme phénotype cible pour identifier des biomarqueurs structurels et fonctionnels.

## Papiers
- [[Bethlehem2020]] — Étude de l'hétérogénéité de l'épaisseur corticale dans le trouble du spectre de l'autisme.
- [[Bethlehem2022]] — Inclus dans les comparaisons trans-diagnostiques cas-témoins de centiles.
- [[Chen2021]] — Exemple où les déviations d'épaisseur corticale corrèlent avec la sévérité des symptômes (Zabihi 2019).
- [[Dorfschmidt2025]] — Variabilité accrue des centiles d'asymétrie chez les patients autistes (asymétrie fluctuante).
- [[Kumar2023]] — Le risque d'ASD conféré par les CNV est corrélé à leurs tailles d'effet sous-corticales ; ASD forme un cluster distinct.
- [[Rutherford2022]] — Groupe clinique inclus dans l'échantillon transdiagnostique (données ABIDE notamment).

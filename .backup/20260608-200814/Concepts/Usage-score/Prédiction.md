# Prédiction

## C'est quoi
Usage des scores de déviation ou de leurs dérivés pour prédire un diagnostic, un phénotype clinique ou une variable d'intérêt, typiquement via des modèles supervisés (régression logistique, SVM, etc.).

## Papiers
- [[Elleaume2025]] — Classification AD vs HC par SVC sur les z-scores (AUC≈0,86, robuste dès ~15 sujets).
- [[Ge2024]] — Les z-scores régionaux alimentent un SVC (classification diagnostique, AUC 0,63) et une régression ridge (sévérité PANSS).
- [[Holz2023]] — Régression logistique des 20 composantes LICA prédisant le diagnostic DBD (23 % de variance).
- [[Lawn2024]] — Régression logistique sur le score résumé pour discriminer patients/sains (AUC jusqu'à 0,75).
- [[Mito2025]] — Prédiction individualisée du diagnostic et de l'issue thérapeutique comme finalité visée.
- [[Parker2025]] — Les centiles régionaux prédisent la performance cognitive et la captation tau PET temporale.
- [[Rutherford2022b]] — Analyse en aval : modélisation prédictive comportementale à partir des déviations.
- [[Verdi2024]] — Analyse de survie de Cox : la hausse du tOC sur 12 mois prédit le risque de conversion MCI→AD (HR=1,09).
- [[Xie2025]] — Classifieurs supervisés sur les W-scores : TLE vs témoins (AUC 0,77), liberté de crise (AUC 0,64).
